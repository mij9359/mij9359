#### **2️⃣ 평가 시간표 엑셀** (날짜별 시트 분할)

```
시트 1: "1일차 (07/09)"
┌──────────────────────────────────────────┐
│ 시간      | 팀명 | 학교명 | ...           │
├──────────────────────────────────────────┤
│ 10:00    │      │ (팀1, 팀2, 팀3 병합)  │
│ 10:00    │ 팀1  │ ...                   │
│ 10:08    │ 팀2  │ ...                   │
│ 10:16    │ 팀3  │ ...                   │
│ 10:24    │      │ [BREAK 5분 병합]      │
│ 10:29    │ 팀4  │ ...                   │
│ ...      │ ...  │ ...                   │
└──────────────────────────────────────────┘

시트 2: "2일차 (07/10)"
┌──────────────────────────────────────────┐
│ ...
└──────────────────────────────────────────┘
```

### 복잡성

```
❌ 단순 반복 (한 시트에 모든 데이터):
   → 파일이 너무 큼 (650KB+)
   → 행이 1만 개 이상 (엑셀 성능 저하)

❌ 병합셀 없이 단순 나열:
   → 시간표가 한눈에 안 들어옴
   → 운영팀이 알아보기 어려움

✅ 해결책:
   ├─ 날짜별로 시트 자동 분할 (7~10개 시트)
   ├─ 연속 발표 슬롯만 세로로 병합
   ├─ BREAK/LUNCH는 별도 행으로 강조
   └─ 인쇄 가능한 레이아웃
```

```java
public ByteArrayInputStream generateBeforeAssignedPointExcel(
        ScoreCorrectionResultResVO data) {
    try (Workbook workbook = new XSSFWorkbook()) {
        Sheet sheet = workbook.createSheet("보정 점수 결과");
        
        CellStyle titleStyle = createTitleStyle(workbook);
        CellStyle headerStyle = createHeaderStyle(workbook);
        CellStyle dataStyle = createCenterStyle(workbook);
        
        int rowIdx = 0;
        
        // ═══════════════════════════════════════════════════
        // 섹션 1: 전체 통계 요약
        // ═══════════════════════════════════════════════════
        
        // 제목 (병합셀)
        Row titleRow = sheet.createRow(rowIdx++);
        Cell titleCell = titleRow.createCell(0);
        titleCell.setCellValue("전체 통계 요약");
        titleCell.setCellStyle(titleStyle);
        sheet.addMergedRegion(new CellRangeAddress(
            rowIdx - 1, rowIdx - 1, 0, 2)); // ← 3개 열 병합
        
        // 헤더
        Row headerRow = sheet.createRow(rowIdx++);
        String[] headers1 = {"전체 원점수 평균", "보정 전체 평균", "보정 전체 표준편차"};
        for (int i = 0; i < headers1.length; i++) {
            Cell cell = headerRow.createCell(i);
            cell.setCellValue(headers1[i]);
            cell.setCellStyle(headerStyle);
        }
        
        // 데이터
        Row dataRow = sheet.createRow(rowIdx++);
        setCellValue(dataRow, 0, data.getTotalRawAvg(), dataStyle);
        setCellValue(dataRow, 1, data.getAdjOverallAvg(), dataStyle);
        setCellValue(dataRow, 2, data.getAdjOverallStdDev(), dataStyle);

        // 섹션 2: 권역별 보정 요약

        // 헤더
        Row regionalHeader = sheet.createRow(rowIdx++);
        String[] headers2 = {"구분", "평균", "보정계수"};
        for (int i = 0; i < headers2.length; i++) {
            Cell cell = regionalHeader.createCell(i);
            cell.setCellValue(headers2[i]);
            cell.setCellStyle(headerStyle);
        }
        
        // 권역별 데이터
        if (data.getRegionalSummaries() != null) {
            for (RegionalSummaryVO summary : data.getRegionalSummaries()) {
                Row row = sheet.createRow(rowIdx++);
                setCellValue(row, 0, summary.getRegionName(), dataStyle);
                setCellValue(row, 1, summary.getAverage(), dataStyle);
                setCellValue(row, 2, summary.getCorrectionFactor(), dataStyle);
            }
        }
        
        // 빈 행
        rowIdx++;
        
        // ═══════════════════════════════════════════════════
        // 섹션 3: 상세 팀 리스트 (31컬럼!)
        // ═══════════════════════════════════════════════════
        
        Row titleRow3 = sheet.createRow(rowIdx++);
        Cell titleCell3 = titleRow3.createCell(0);
        titleCell3.setCellValue("상세 팀 리스트");
        titleCell3.setCellStyle(titleStyle);

            rowIdx - 1, rowIdx - 1, 0, 28)); // ← 29개 열 병합

            "현재 트랙명", "배정된 트랙명", "권역 및 아이템", "팀 번호", "회원 번호",

        }

        if (data.getTeamList() != null) {
            for (AssignedPointListResVO team : data.getTeamList()) {
                Row row = sheet.createRow(rowIdx++);
                int colIdx = 0;
                
                setCellValue(row, colIdx++, team.getOriginalTrackName(), dataStyle);
                setCellValue(row, colIdx++, team.getAssignedTrackName(), dataStyle);
                setCellValue(row, colIdx++, team.getPgName(), dataStyle);
                // ... (26개 더)
                setCellValue(row, colIdx++, team.getFinalRank(), dataStyle);

        }
        
        // 컬럼 너비 자동 조정
        for (int i = 0; i < teamHeaders.length; i++) {
            sheet.autoSizeColumn(i);
        }
        
        // 워크북을 바이트로 변환
        ByteArrayOutputStream out = new ByteArrayOutputStream();
        workbook.write(out);
        return new ByteArrayInputStream(out.toByteArray());
        
    } catch (IOException e) {
        log.error("보정 점수 결과 엑셀 생성 실패", e);
        throw new RuntimeException("엑셀 파일 생성 실패", e);

}
```

### Part 2: 시간표 엑셀 (날짜별 시트 + 동적 병합)

```java
public ByteArrayInputStream getTimetableExcel(
        List<EvaluationTimetableSlotVO> slots) {
    try (Workbook workbook = new XSSFWorkbook()) {
        
        CellStyle timeStyle = createTimeStyle(workbook);
        CellStyle evaluationStyle = createEvaluationStyle(workbook);
        CellStyle breakStyle = createBreakStyle(workbook, Color.YELLOW);
        CellStyle lunchStyle = createBreakStyle(workbook, Color.LIGHT_GREEN);
        
        // ═══════════════════════════════════════════════════
        // 날짜별로 슬롯 그룹화
        // ═══════════════════════════════════════════════════
        
        Map<LocalDate, List<EvaluationTimetableSlotVO>> slotsByDate = 
            slots.stream()
                .collect(Collectors.groupingBy(
                    EvaluationTimetableSlotVO::getEvaluationDate));
        
        // 날짜 순서대로 정렬
        List<LocalDate> dates = slotsByDate.keySet().stream()
            .sorted()
            .collect(Collectors.toList());
        
        // ═══════════════════════════════════════════════════
        // 각 날짜별 시트 생성
        // ═══════════════════════════════════════════════════
        
        for (LocalDate date : dates) {
            // 시트명: "1일차 (07/09)"
            String sheetName = String.format("%d일차 (%s)",
                dates.indexOf(date) + 1,
                date.format(DateTimeFormatter.ofPattern("MM/dd")));
            
            Sheet sheet = workbook.createSheet(sheetName);
            List<EvaluationTimetableSlotVO> daySlots = slotsByDate.get(date);
            
            int rowIdx = 0;
            
            // 헤더
            Row headerRow = sheet.createRow(rowIdx++);
            String[] headers = {"시간", "팀명", "학교명", "발표자", "연락처"};
            for (int i = 0; i < headers.length; i++) {
                Cell cell = headerRow.createCell(i);
                cell.setCellValue(headers[i]);
                cell.setCellStyle(createHeaderStyle(workbook));
            }
            
            // ───────────────────────────────────────────────
            // 핵심: EVALUATION 슬롯 그룹을 추적하며 병합
            // ───────────────────────────────────────────────
            
            int groupStartRow = -1;
            LocalTime lastGroupStartTime = null;
            
            for (EvaluationTimetableSlotVO slot : daySlots) {
                Row row = sheet.createRow(rowIdx);
                
                // 슬롯 타입에 따라 스타일 결정
                CellStyle currentStyle;
                if (slot.getSlotType() == SlotType.LUNCH) {
                    currentStyle = lunchStyle;
                } else if (slot.getSlotType() == SlotType.BREAK) {
                    currentStyle = breakStyle;
                } else {
                    currentStyle = evaluationStyle;
                }
                
                // 시간 입력
                Cell timeCell = row.createCell(0);
                timeCell.setCellValue(slot.getStartTime().toString());
                timeCell.setCellStyle(timeStyle);
                
                // 타입별 처리
                if (slot.getSlotType() == SlotType.EVALUATION) {
                    
                    // EVALUATION인 경우, 그룹 시작 추적
                    if (groupStartRow == -1) {
                        groupStartRow = rowIdx;
                        lastGroupStartTime = slot.getStartTime();
                    }
                    
                    // 팀정보 입력
                    setCellValue(row, 1, slot.getTeamName(), evaluationStyle);
                    setCellValue(row, 2, slot.getTeamSchool(), evaluationStyle);
                    setCellValue(row, 3, slot.getTeamLeader(), evaluationStyle);
                    setCellValue(row, 4, slot.getTeamPhone(), evaluationStyle);
                    
                } else {
                    // BREAK/LUNCH: 기존 EVALUATION 그룹 종료 → 병합
                    if (groupStartRow != -1) {
                        // 시간 컬럼만 병합 (팀 정보 없음)
                        sheet.addMergedRegion(new CellRangeAddress(
                            groupStartRow, rowIdx - 1, 0, 0));
                        RegionUtil.setBorderBottom(BorderStyle.THIN, 
                            new CellRangeAddress(
                                groupStartRow, rowIdx - 1, 0, 0), sheet);
                        
                        groupStartRow = -1;
                    }
                    
                    // BREAK/LUNCH 행
                    String label = (slot.getSlotType() == SlotType.LUNCH) 
                                 ? "점심시간" : "휴식시간";
                    setCellValue(row, 1, label, currentStyle);
                    sheet.addMergedRegion(new CellRangeAddress(
                        rowIdx, rowIdx, 1, 4)); // 4개 열 병합
                }
                
                rowIdx++;
            }
            
            // 마지막 EVALUATION 그룹 병합
            if (groupStartRow != -1) {
                sheet.addMergedRegion(new CellRangeAddress(
                    groupStartRow, rowIdx - 1, 0, 0));
            }
            
            // 컬럼 너비 설정
            sheet.setColumnWidth(0, 2500); // 시간
            sheet.setColumnWidth(1, 3500); // 팀명
            for (int i = 2; i < 5; i++) {
                sheet.autoSizeColumn(i);
            }
            
            // 인쇄 설정
            sheet.setFitToPage(true);
            sheet.getPrintSetup().setPaperSize(PrintPaper.A4);
            sheet.setHorizontallyCenter(true);
        }
        
        ByteArrayOutputStream out = new ByteArrayOutputStream();
        workbook.write(out);
        return new ByteArrayInputStream(out.toByteArray());
        
    } catch (IOException e) {
        log.error("시간표 엑셀 생성 실패", e);
        throw new RuntimeException("엑셀 생성 실패", e);
    }
}
```

## ⚠️ 마주친 문제점 & 해결

### **Problem 1: 셀 병합과 스타일이 겹침**

**증상:**
```
sheet.addMergedRegion(범위);
cell.setCellStyle(스타일);

→ 병합된 범위 중 일부 셀의 스타일이 적용 안 됨
→ 특히 테두리가 불완전하게 나타남 ❌
```

**해결:**
```java
// 병합 후 테두리 명시적으로 설정
sheet.addMergedRegion(cellRange);
RegionUtil.setBorderBottom(BorderStyle.THIN, cellRange, sheet);
RegionUtil.setBorderTop(BorderStyle.THIN, cellRange, sheet);
RegionUtil.setBorderLeft(BorderStyle.THIN, cellRange, sheet);
RegionUtil.setBorderRight(BorderStyle.THIN, cellRange, sheet);
```

### **Problem 2: 한 셀 안에 여러 색상 (제목 일부만 파란색)**

**증상:**
```
"1일차 (07/09)" 에서 "1일차"는 검정색, "(07/09)"는 파란색

Cell에는 하나의 CellStyle만 적용 가능 ❌
```

**해결:**
```java
// RichTextString 사용
XSSFRichTextString richText = new XSSFRichTextString("1일차 (07/09)");

// 폰트 1: 검정색
XSSFFont blackFont = workbook.createFont();
blackFont.setColor(IndexedColors.BLACK.getIndex());
richText.applyFont(0, 3, blackFont); // "1일차"

// 폰트 2: 파란색
XSSFFont blueFont = workbook.createFont();
blueFont.setColor(IndexedColors.BLUE.getIndex());
richText.applyFont(3, 8, blueFont); // "(07/09)"

cell.setCellValue(richText);
```

### **Problem 3: 연속 EVALUATION 슬롯 병합**

**증상:**
```
팀1 10:00-10:08
팀2 10:08-10:16  ← 이 3개를 "시간" 컬럼만 병합하고 싶음
팀3 10:16-10:24

하지만 중간에 BREAK가 들어오면?
팀1 10:00
팀2 10:08
[BREAK] 10:16  ← 여기서 그룹 종료
팀3 10:24

→ 그룹을 추적하면서 BREAK 만나면 병합해야 함
```

**해결:**
```java
int groupStartRow = -1;

for (slot : daySlots) {
    if (slot.type == EVALUATION) {
        if (groupStartRow == -1) {
            groupStartRow = rowIdx; // 그룹 시작
        }
    } else { // BREAK/LUNCH
        if (groupStartRow != -1) {
            // 기존 그룹 병합
            sheet.addMergedRegion(new CellRangeAddress(
                groupStartRow, rowIdx - 1, 0, 0));
            groupStartRow = -1; // 리셋
        }
    }
    rowIdx++;
}

// 마지막 그룹
if (groupStartRow != -1) {
    sheet.addMergedRegion(new CellRangeAddress(
        groupStartRow, rowIdx - 1, 0, 0));
}
```

### **Problem 4: 파일 크기 폭발**

**증상:**
```
300팀 × 31컬럼을 단일 시트에 생성
→ 파일 크기: 5MB+
→ 엑셀 열기 시간: 10초+

시간표 일자별 미분할 경우
→ 슬롯 1000개 × 5컬럼 × 1시트
→ 파일 크기: 2MB, 시간: 5초
```

**해결:**
```java
// 보정 점수: 단일 시트 (불가피)
// 시간표: 날짜별 다중 시트로 자동 분할
Map<LocalDate, List<Slot>> slotsByDate = slots.stream()
    .collect(groupingBy(Slot::getDate));

for (LocalDate date : dates) {
    workbook.createSheet("N일차");
}
// → 파일 크기 50% 감소, 로딩 속도 10배 증가
```

## 🎓 배운 점

### 1. 병합셀과 스타일의 관계

> **"병합 후에 테두리를 명시적으로 그어야 한다"**

- POI는 병합된 범위의 스타일을 완벽하게 복제하지 않음
- **RegionUtil 유틸로 테두리를 수동 설정** 필수

### 2. RichTextString의 강력함

> **"한 셀에 여러 포맷을 적용할 수 있다"**

- 제목에서 날짜만 강조 색상
- 주의사항: 문자 위치(0, 3, 8)를 정확히 계산해야 함

### 3. 대용량 엑셀은 시트 분할이 핵심

> **"날짜/카테고리별로 시트를 나누면 파일 크기와 로딩 속도 모두 개선된다"**

- 단일 시트 vs 다중 시트: 5배 차이
- 사용자 UX도 향상 (원하는 시트만 열기 가능)

### 4. 프린트 친화적 레이아웃

```java
sheet.setFitToPage(true);
sheet.getPrintSetup().setPaperSize(PrintPaper.A4);
sheet.setHorizontallyCenter(true);
```

→ 심사위원회 발표 자료로 바로 인쇄 가능!
